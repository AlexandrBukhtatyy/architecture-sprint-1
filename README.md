# Спринт 1 - практика

## Описание
Краткое описание проекта.

## Настройка окружения
Необходимо установить node актуальной версии

Node можно устаноить через nvm

## Локальный запуск
```
npm i           // установка зависимостей 
npm run start   // запуск проекта
```
